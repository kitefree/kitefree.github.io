11. Physical Layer

物理層拓璞

Network Cabling 網路線

只要可以傳播數據的我們都可以稱為是網線

Ethernet Cabling

最常見的為雙絞線

Twisted Pair Cable

  • UTP
  • STP
    • 減少電磁波的干擾

STP 相對貴一些
但是現在的UTP 其實不太影響，除非是很特殊的環境

UTP Categories

CAT 是指Categories

UTP Connector

凹的是JACK
凸的是PLUG

RJ-45 接線標準


現在常用的是T568B
白橙橙白綠藍白藍綠白棕棕
RJ-11 電話線

Straight-Through Cable

不同種類的設備的連線

  • Host to Switch
  • Router to Switch
  • Host to Router

Crossover Cable

同種設備的連線

  • Host to Host
  • Switch to Switch
  • Router to Router

不過目前是可以直接使用Straight-Through Cable ,系統偵測如果線序是不一致的，會進行跳線轉換
Automatic Medium-Dependent Interface Crossover(Auto-MDIX)

Rolled Cable

Console cable 就是用這種線序

Fiber Optic

  • Single-mode 單模
    • 遠距離傳輸
    • 較貴
  • Multi-mode 多模
    • 近距離傳輸

光纖使用的光
請不要用眼睛直接看裡面的光

  • LED
  • 激光
    • 激光對視網模會造成損害

OM (Optical Multimode)

大部分會使用OM3

  • OM1
  • OM2
  • OM3
  • OM4

Fiber Connector

光纖的接口叫connect
最常見的是SC、LC
比較舊的設備是用SC
現在比較常用的是LC

SFP Transceiver

當有了線有了頭，還要一個東西就是這個，這個東西叫模塊
買交接機的時候要注意一下後面是固定的RJ-45接口呢還是空的位置讓你插
SFP 也可以拿來接RJ-45的哦!!

Transceiver Code

太多種了，只講以下兩種

  • SR stnads for Short Reach(10GBase-SR)
    • Short Range
    • Almost always multi-mode (多模)
  • LR stands for Long Reach(10GBase-LR)
    • Long Range
    • Either multi-mode or single-mode(單模)

當在買模塊的時候，要考慮線材是多模的還是單模的，再去挑對應的模塊代號

Simple ,HalfDuplex ,Duplex

  • 單工
    • 單行道
    • 只能單向傳輸
  • 半雙工
    • 一個時間點只允一個方向傳輸
  • 全雙工
    • 雙向道
    • 同一個時間點可以同時允許雙向傳輸

如果兩個交換機 之間是 1 Gbps /Full Duplex
那我們在講吞吐量的時候會說有2gbps，但是一樣每一個方向最多是1g
