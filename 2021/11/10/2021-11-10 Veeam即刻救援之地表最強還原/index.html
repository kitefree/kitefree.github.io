6.1. Instant VM Recovery(即時還原)

6.3. Instant VM Recovery

只要透過veeam備份(多種來源，下圖所示)，都可以在vmware 還原

6.4. 雲端備份

6.4.1. 完整還原/精細還原

備份丟到雲端時是完整備份，要還原的時候可以透過veeam挑出您要的檔案進行還原，這個就是所謂的精細還原，下圖綠框清單舉例了精細還原的應用對象

6.5. 將服務搬上雲端

不是只是把檔案搬上去而已，可以做到把服務放到雲端，服務營運在雲端

6.6. 將Cloud服務快速還原至地端

6.7. AWS marketplace

可以在 marketplace 點選這個服務，將這個服務打開。可以在雲端上面備份雲端上面的虛擬機，備份到雲端的Storage Service(S3)

6.8. Azure marketplace

6.9. 還原至地端服務

先將object copy 到地端的backup server ，再透過veeam 工具佈署到你要的環境上，比如vmware 等等。

6.10. 快速還原到地端

也支援快速還原到地端的功能

6.11. 雲跟雲之間互換

從AWS 或 Azure 備份檔案透過地端的Veeam 還原到其它的cloud上
