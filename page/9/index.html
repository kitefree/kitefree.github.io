什麼是負載平衡?

發表於 2021-05-30 更新於 2023-12-31 分類於 負載平衡
400 1 分鐘

1. 什麼是負載平衡?

  • 頻寬管理

    • LAN Load Balance
    • WAN Load Balance
      WAN Load Balance（WLB），一般也稱為Multi-Homing設備。當企業對外線路有兩條以上的時候，WLB就可以擔負起網際網路（Internet）與內部網路（Intranet）之間多條連外線路流量平衡的工作。 LAN Load Balance針對區域網路，設置在封包類別較複雜的節點，有助於調整、控制封包的進出順序。

  • 應用管理

    • Server Load Balance
      Server Load Balance（SLB）是最早發展的一塊，其功能在於將企業內多部伺服器的負載量作平衡，將過大流量轉向相同功能的其他伺服器，讓每一部伺服器工作量維持平均，保持服務不因流量過大而變慢或中斷。SLB主要應用在入口端的網頁分流，透過SLB轉址到各個伺服器，平均分配網路流量。
